Notebooks
=========

Interactive Jupyter notebooks demonstrating nos-utils workflows.

SECOFS prep walkthrough
-----------------------

The ``notebooks/secofs_prep_walkthrough.ipynb`` notebook walks through a
complete SECOFS prep cycle step by step, showing how each processor is
configured and invoked.

To run it:

.. code-block:: bash

   pip install -e ".[full]" jupyter
   jupyter notebook notebooks/secofs_prep_walkthrough.ipynb
